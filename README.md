# 🌸 Mother's Day Gift Guide for Cat Moms

## 📁 你需要上传到 GitHub 的文件

上传这些到你 `thefluffylab` 仓库的**根目录**：

```
momgift.html              ← 主页面
hero3.jpg                 ← 首图（网页用）
hero3-pinterest.jpg       ← Pinterest 专用图
momgift-images/           ← 产品图文件夹（整个上传）
  ├── 01-willow-tree.jpg
  ├── 02-cat-candle.jpg
  ├── 03-smiley-planter.jpg
  ├── 04-tumbler.jpg
  ├── 05-eye-mask.jpg
  ├── 06-card.jpg
  └── 07-gift-box.jpg
```

所有文件名都不会跟你仓库现有的冲突 ✅

---

## 🚀 部署步骤

### 1. 上传文件到 GitHub

1. 进入你的 `thefluffylab` 仓库
2. 点绿色 **"Add file"** → **"Upload files"**
3. 把下面这些拖进去：
   - `momgift.html`
   - `hero3.jpg`
   - `hero3-pinterest.jpg`
   - `momgift-images` 整个文件夹
4. 页面底部 **"Commit changes"** → 点绿色按钮确认

### 2. 访问你的新页面

等 1-2 分钟，打开：
```
https://youyouyu52-lang.github.io/thefluffylab/momgift.html
```
